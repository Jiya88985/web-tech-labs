# lab03-css

This repository contains the CSS lab project structure used for the Web Tech Lab course.

## Structure

- `index.html` — Lab landing page
- `styles/` — Main CSS files
  - `main.css`
  - `typography.css`
  - `components.css`
  - `effects.css`
- `images/` — Assets (backgrounds, gallery, icons)
- `exercises/` — Exercise pages for selector, typography, background, and image practice

## Getting Started

Open `index.html` in your browser to begin.
